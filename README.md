# PEDIDOS ALVARO

App interna liviana para reemplazar el cuaderno físico de faltantes de una ferretería/retail.

## Objetivo

Digitalizar primero el flujo operativo más importante: registrar faltantes desde celular usando productos oficiales y alias, evitando nombres desordenados y dejando preparada la evolución hacia pedidos por proveedor e importación de facturas Excel.

## Ejecutar

Abrir `src/index.html` en un navegador moderno. La Fase 1 no requiere backend ni dependencias externas.

## Probar

```bash
npm test
```
